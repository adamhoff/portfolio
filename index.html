Tower Of Hanoi

A front-end mathematical drag n'drop puzzle game

More Info

Tower of Hanoi was is my first front-end project using HTML, CSS and JavaScript taking a total of 4-5 days to complete. The game consists of three rods and a number of disks of different sizes, which can append onto any rod. The puzzle starts with the disks in a stack in ascending order by size on one rod, the smallest at the top. It can be played with no less than three discs, up to a maximum of eight.

The rules of the game are as follows:

  • Only one disk may be moved at a time.
  • A disk can only be moved if it is the uppermost disk on a stack.
  • No disk may be placed ontop of a smaller disk.

The game is finished once all disks are moved from the first stack, to the third stack.
